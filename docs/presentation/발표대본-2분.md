## 🗣 Q&A 대비 (자주 나오는 질문)

**"AI가 만든 부분 / 본인이 만든 부분은?"** (배점 큼 — 솔직하게)
> "피드·캐시 같은 골격은 AI Agent가 생성했고, 저는 **error 152를 실기기에서 재현하고, 라이브러리 소스에서 origin이 host로도 쓰인다는 점을 찾아내 youtube-nocookie 해법을 검증·채택**했습니다. AI가 틀린 진단(합성 불가 가설)도 데이터로 폐기했습니다."

**"왜 Flutter인가요?"**
> "단일 코드베이스로 세로 피드를 빠르게 구현하기 좋고, ADR-0001에 근거를 정리했습니다."

**"이 화면의 데이터는 어디서 오나요?"**
> "YouTube Data API → Repository에서 Firestore에 캐싱 → Provider로 피드에 전달됩니다. 같은 토픽은 캐시 우선이라 할당량을 아낍니다."

**"지금 가장 큰 문제는?"**
> "콘텐츠 품질 튜닝입니다. 학습 키워드+relevance로 예능을 줄였지만 더 다듬는 중이고, 안 되면 Education 카테고리 필터를 추가할 계획입니다."

**모르는 질문이 나오면**
> "그 부분은 정확히 모릅니다. 추측으로는 ~이지만, 발표 후 확인해서 알려드리겠습니다."
> → 모르는 걸 인정하는 게 거짓 답변보다 **가점**.
